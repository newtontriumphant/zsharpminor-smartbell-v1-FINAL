# zsharpminor-smartbell-v1-FINAL
ESP32-powered smart doorbell with smart AI-powered detection, that automatically captures a picture that is sent to a Discord webhook via WiFi when the doorbell is rung, which triggers an integration that can set off a custom ringtone on a mobile app. Features near-waterproof PETG/TPU case and a temperature sensor that automatically reports T/H. &lt;3
